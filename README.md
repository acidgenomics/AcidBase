Low-level base functions imported by [Acid Genomics][] packages.

## Installation

This is an [R][] package.

### [R][] method

```r
install.packages(
    pkgs = "AcidBase",
    repos = c(
        "https://r.acidgenomics.com",
        getOption("repos")
    )
)
```

[acid genomics]: https://acidgenomics.com/
[r]: https://www.r-project.org/
